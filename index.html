Etiquetas de estructura de texto

Uso de etiqueta "p":

Esto es un parrafo

Uso de etiqueta "pre":

 Esto es un texto preformateado, mantiene la estructura y formato del código fuente.

Uso de etiqueta "blockquote":

Permite especificar una sección citada desde otra fuente.

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

Uso de etiqueta "q":

Define una cita en línea de un documento HTML.

WWF's goal is to: Build a future where people live in harmony with nature. We hope they succeed.

Uso de etiqueta "cite":

Define el título de una obra citada de un documento HTML.

The Scream by Edward Munch. Painted in 1893.

Uso de etiqueta "abbr":

Define una abreviatura o un acrónimo de un documento HTML.

El INA es una Institución autónoma que promueve la formación y capacitación profesional.

Uso de etiqueta "dfn":

Define una definición de un término de un documento HTML.

HTML es el lenguaje de marcado estándar para el desarrollo de páginas web.

Uso de etiqueta "code":

Define un fragmento de código de un documento HTML.

"La etiqueta button define un botón presionable."

"La propiedad background-color define el color de fondo de un elemento."

Uso de etiqueta "kbd":

Define una entrada de teclado de un documento HTML.

Presione Ctrl + C para copiar texto.

Presione Cmd + V para pegar el texto copiado.

Uso de etiqueta "strong":

Define un texto fuerte de un documento HTML.

Esto es un texto importante!

Uso de etiqueta "i":

Define un texto en cursiva de un documento HTML.

Este es un texto en cursiva.

Uso de etiqueta "em":

Define un texto enfático de un documento HTML.

Preste atención a este texto.

Uso de etiqueta "mark":

Resalta o marca un fragmento de texto.

Este es un texto marcado.

Uso de etiqueta "small":

Muestra información secundaria o aclaratoria.

Esto es un texto de ejemplo (Esto es un texto secundario)

Uso de etiqueta "del":

Indica texto eliminado o reemplazado.

"Antes: ₡12 000 Ahora: ₡10 000"

Uso de etiqueta "ins":

Indica texto insertado o agregado.

"Se agregó este producto a la canasta."

Uso de etiqueta "sub":

Muestra texto en subíndice (debajo de la línea base).

"Fórmula del agua: H2O"

Uso de etiqueta "sup":

Muestra texto en superíndice (encima de la línea base).

"Ecuación: x2 + y2 = r2"

Uso de etiqueta "time":

Representa una fecha u hora con un valor legible por máquinas.

Fecha: 23 de septiembre de 2025

Hora: 9:15 p.m.

Asocia un valor de datos a un texto visible.
